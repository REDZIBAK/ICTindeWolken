Voor mijn eerste project deed ik mee met de anderhalve meter project.
Het doel was om een manier te vinden om anderhalve meter te kunnen detecteren.

Om afstand te kunnen detecteren heb ik met de micro:bit gewerkt.
Een microbit is een kleine chip met een scherm van 25 pixels en twee knoopjes.
Een microbit heeft veel meer maar voor deze project hebben we de radios nodig.

De microbits gaan radiosignalen naar elkaar zenden en daarmee kunnen we detecteren hoe dicht ze bij elkaar staan.
Als de bicrobbits binnen dezelfde radiogroep zitten, dan zullen ze de signalen van andere microbits herkennen.

Hierboven zie je een code die ervoor zorgt dat de microbits steeds signalen naar elkaar zenden.
Als de microbit een signaal ontvangt met een signaalsterkte groter dan -75, telt hij 1 bij.
Je krijgt dan ook een "X" te zien op je scherm.
Als je dan op knop "A" drukt, krijg je hoeveel seconden je geen afstand hebt gehouden.
Als je wel afstand houdt zie je een ":)" op je scherm.

Als volgt willen we een ID maken voor elke microbit en detecteren met hoeveel mensen (ID's) we geen afstand hebben gehouden.
We willen een lijst maken met daarin het aantal ID die we zijn tegengekomen binnen anderhalve meter.
We willen ook niet dat we dezelfde ID twee keer in onze lijst krijgen.
Als de ID niet in de lijst staat, zal hij een index van -1 teruggeven.
Anders geeft hij een getal >= 0.
Als je op knopje "B" drukt krijg je de lengte van de lijst.

De microbit kan worden gebruikt in gesloten ruimtes om bij te houden of je afstand houdt van anderen.
Je kunt vervolgens mensen belonen die wel afstand houden van anderen.
Zo'n functie kan ook worden toegevoegd aan een corona-app die ook andere functies bevat zoals een soort Google Maps
om drukke plekken te kunnen vermijden.

In deze project heb ik geleerd hoe je een microbit-code kan schrijven en op de microbit kan zetten.
Ik heb ook geleerd zelf aan werk te gaan.

Een probleem dat ik tegenkwam was de signaalsterkte die af en toe verschilde zonder reden.
Een ander probleem waar we een oplossing voor hebben bedacht, was dat de teller niet seconden telt maar signalen.
De microbit zond meer dan 1 signaal per seconde uit en daarvoor moest ik een seconde wachttijd instellen.
